# 2 - Navigation

## Objetivo

O objetivo é ter um componente de navegação de barra lateral que seja fixo nas paginas de Dashboard, Clientes e Projetos. Paginas seguintes e formulário ele deve sumir.

## Definições

- o componente é o navigation rail do [material](https://m3.material.io/components/navigation-rail/overview)
- A pagina de Dashboard deve ter o nome de Dashboard e o ícone ```<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:opsz,wght,FILL,GRAD@20..48,100..700,0..1,-50..200&icon_names=space_dashboard" />```
- A pagina de Clientes deve ter o nome de Clientes e o ícone ```<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:opsz,wght,FILL,GRAD@20..48,100..700,0..1,-50..200&icon_names=identity_platform" />```
- A pagina de Projetos deve ter o nome de Projetos e o ícone ```<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:opsz,wght,FILL,GRAD@20..48,100..700,0..1,-50..200&icon_names=assignment" />```
